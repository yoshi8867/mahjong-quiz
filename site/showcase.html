34종 전체 타일 — 유니코드 방식 vs CSS 커스텀 방식

만수 1-9, 통수 1-9, 삭수 1-9, 자패 7종(동남서북·백발중) = 34종을 같은 순서로 두 줄 비교.

유니코드 글리프 (U+1F000~U+1F021)
CSS 커스텀 타일 (아이보리 몸체 + 그림자)

※ 테두리 마스크와 GL-MahjongTile SVG 모드는 DOM 줄에 재현하지 않습니다 — 위 캔버스 미리보기에서 확인하세요.
